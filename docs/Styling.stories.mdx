# Custom Styling

---

The Nodes, Edges, Arrows and Ports can all be overriden to pass custom attributes
to them to allow for custom styling. Below is an example of some of the possibilities
you can achieve.

<br />
<div style={{ height: 500, width: 700, position: 'relative', margin: '0 auto', background: 'white' }}>
  <Story id="demos-basic--styling" />
</div>
<br />

In the above example, we override the `node`, `edge` and `arrow` properties
in the `Canvas` component as shown below (abbreviated for demonstration purposes):

```jsx
import { Canvas, Node, Edge, Port, MarkerArrow } from 'reaflow';

export const CustomCanvas: FC = () => (
  <Canvas
    nodes={[]}
    edges={[]}
    node={
      <Node
        style={{ stroke: '#1a192b', fill: 'white', strokeWidth: 1 }}
        label={<Label style={{ fill: 'black' }} />}
        port={<Port style={{ fill: 'blue', stroke: 'white' }} rx={10} ry={10} />}
      />
    }
    arrow={<MarkerArrow style={{ fill: '#b1b1b7' }} />}
    edge={<Edge className="edge" />}
  />
);
```

This allows us to set custom properties on these components or even
override them all together with our own component. This allows for
total control over the child components without loosing all the "magic"
under the hood that controls them.
